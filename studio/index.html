Design an ASS preset visually. Export .ts → drop into packs/premium/ or load via --preset-file. 100% offline, runs from file://.

Typography
Colors (3-state karaoke)

"Past" is the default colour for unspoken text in 2-state mode. Upcoming reads as "not yet spoken" only when the CLI gets --upcoming.

Outline & shadow (default state)

BorderStyle 4 paints a solid bar across the full caption line — the active-state colour swap then changes the bar bg. Use for news-ticker / pill-shadow looks.

Active word effects

Defaults = Hormozi-style (amber fill + black outline + scale 1.15). Tick "fill → dark" + "outline → highlight" for the pill stamp look. Avoid both highlight-outline + amber fill — same colour everywhere reads as a blob.

Position

65 = cross-platform safe zone (avoids TT/IG/YT chrome).
